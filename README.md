# Tetris
 
This a cross platform terminal game Tetris.
### High points
#### 1.CrossPlatform
Supoort Linux/Macos/Windows
#### 2.MultiThread
High performance
### How to build
 
On Linux/Unix : 
 
```
make Tetris
```
On Windows : you should install mingw(POSIX) to build

```
mingw32-make Tetris
```
or add source files and head files into VidsualStudio(you can use msvc too)
